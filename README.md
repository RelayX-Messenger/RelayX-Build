# Installing Release v0.1 of RelayX

1. Go to the releases and download the latest one (v0.1) and extract it to C:\Program Files\RelayX
2. Program Files does not allow executables to do malware behavior like extracting to the working dir (Program Files blocks it).
3. If unsure, Extract to a VM and let virustotal analyze all executables individually.  
4. Launch the C:\Program Files\RelayX\RelayX.exe and use FastAPI's SwaggerUI for using endpoints.
